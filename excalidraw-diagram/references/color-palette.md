# Color Palette — Rathish Brand Style

**Single source of truth for all diagram colors.** Customized for Rathish's dark theme design system.

| Tertiary | `#c4b5fd` | `#4c1d95` |

## Text Colors (Hierarchy)

| Level | Color | Use For |
|-------|-------|---------|
| Title | `#8b5cf6` | Section headings, major labels |
| Subtitle | `#a78bfa` | Subheadings, secondary labels |
| Body/Detail | `#94a3b8` | Descriptions, annotations |
| On light fills | `#1e1b4b` | Text inside light shapes |
| On dark fills | `#f8fafc` | Text inside dark shapes |
